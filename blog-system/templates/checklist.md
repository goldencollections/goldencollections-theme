# Template: Checklist

## Purpose

Help buyers plan a purchase or preparation sequence without missing components.

## Ideal Use Case

Use for festival preparation, arangetram preparation, full alankaram, dance performance packing, or temple event planning.

## Section Structure

1. Planning situation introduction
2. Checklist overview
3. Core items
4. Optional items
5. Sizing or compatibility checks
6. Timing and shipping considerations
7. FAQ
8. Next action

## Word Count Range

700-1,200 words.

## Schema Type

`Article` or `BlogPosting`; `FAQPage` when visible.

## Internal Link Requirements

Link each major checklist group to the most relevant collection where possible.

## FAQ Count

4-6 questions.

## EEAT Signals

Use confirmed seasonal timing and shipping facts. Mark festival-specific ritual details as `Needs confirmation` if not sourced.

Topic: Varalakshmi Alankaram Checklist

- When to start planning for May-August demand

- Size and compatibility checks
- Shipping timing for India and international customers
- FAQ
- Shop Varalakshmi jewellery and decoration
