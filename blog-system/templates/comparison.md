# Template: Comparison

## Purpose

Help buyers understand differences between two or more options.

## Ideal Use Case

Use when customers confuse ranges, names, materials, use cases, or regional terms.

## Section Structure

1. Buyer confusion introduction
2. Quick comparison table
3. Option A explained
4. Option B explained
5. When to choose each
6. Price, material, fit, or use considerations
7. Regional vocabulary
8. FAQ
9. Next action

## Word Count Range

800-1,300 words.

## Schema Type

`Article` or `BlogPosting`; add `FAQPage` when visible.

## Internal Link Requirements

Link to each relevant collection or guide being compared.

## FAQ Count

4-6 questions.

## EEAT Signals

Use specific confirmed facts. For real kemp comparisons, include confirmed price ranges and customer-facing `Kemp stones` or `Kempu stones` language.

## Example Outline

Topic: Real Kemp vs Regular Bharatanatyam Jewellery

- H1: Real Kemp vs Regular Bharatanatyam Jewellery: Which Should You Choose?
- Quick comparison table
- What regular Bharatanatyam jewellery is best for
- What real kemp jewellery is best for
- Price range difference
- Material and finish difference
- Which is better for arangetram, class, or repeated performances
- FAQ
- Shop or compare collections
