# Golden Collections Content Operating System

This folder defines the research, briefing, drafting, review, Shopify packaging, publishing, and repurposing workflow for Golden Collections content.

It is not a generic blog generator. It is a Golden Collections-specific operating system for creating useful, culturally accurate, conversion-supporting content for Bharatanatyam jewellery, Kuchipudi jewellery, real kemp jewellery, deity jewellery, Varalakshmi alankaram, and related regional search terms.

## Source Of Truth

Read `knowledge-base/wiki/` before creating any public-facing content.

Core files:

- `knowledge-base/wiki/business-entity.md`
- `knowledge-base/wiki/search-entity-map.md`
- `knowledge-base/wiki/content-roadmap.md`
- `knowledge-base/wiki/real-kemp-jewellery.md`
- `knowledge-base/wiki/deity-compatibility-model.md`
- `knowledge-base/wiki/open-questions.md`

Do not publish uncertain claims. Mark uncertain facts as `Needs confirmation` and flag them for owner review.

## Entry Points

### On-Demand

Use this when the owner provides a topic, for example:

`Real Kemp Jewellery for Arangetram`

Workflow:

1. `00-topic-intake.md`
2. `01-research.md`
3. `02-angle-selection.md`
4. `03-brief.md`
5. `04-draft.md`
6. `05-eeat-review.md`
7. `06-shopify-package.md`
8. Owner approval
9. `07-publish-shopify.md`
10. `08-repurpose.md`

### Calendar-Driven

Use this when the system should suggest what to write next.

Start with `knowledge-snapshot.md`, then review:

- current season
- GSC queries and gaps
- product and collection priorities
- already-published content
- open questions that can become useful content

Then continue from `02-angle-selection.md`.

## Stage Gates

Every stage creates an output that can be reviewed before continuing:

- research: `outputs/research/`
- brief: `outputs/briefs/`
- draft: `outputs/drafts/`
- Shopify-ready package: `outputs/shopify-ready/`
- repurpose package: `outputs/repurpose/`

Do not skip the brief stage for important content. The brief is where the content earns its direction.

## Approval Rule

Nothing is posted to Shopify without owner approval.

The publishing step must create Shopify content as `draft`, not published. Final live publishing remains an owner decision inside Shopify admin.

## Naming Convention

Use lowercase handles with dates when saving stage files:

`2026-05-13-real-kemp-jewellery-arangetram.md`

Use the same handle through research, brief, draft, Shopify package, and repurpose files so the work is easy to trace.

## Quality Standard

A finished piece must:

- answer a real buyer question
- support a product, collection, size decision, or trust decision
- use regional vocabulary naturally
- include internal links
- avoid generic fashion copy
- include accurate Golden Collections trust facts where relevant
- include metadata and FAQ guidance
- pass the EEAT review checklist before Shopify packaging
