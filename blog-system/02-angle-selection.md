# 02 Angle Selection

Use this step after research.

The goal is to choose the most useful and commercially sensible content format for the topic.

## Candidate Angles

Score each relevant angle:

- collection support page
- buying guide
- size and fit guide
- seasonal guide
- checklist
- comparison
- use case guide
- diaspora buying guide
- glossary page
- history and explainer

## Scoring Model

Use the table below. Score only from the available research and wiki facts.

| Factor | Range | Meaning |
| --- | ---: | --- |
| Commercial intent | 0-3 | Drives product discovery or purchase |
| Current season fit | 0-2 | Fits Varalakshmi, arangetram, festival, or known shopping window |
| Collection or product support | 0-2 | Directly supports an active collection or product family |
| GSC opportunity | 0-2 | Query has search presence but no strong page yet |
| Margin or stock priority | 0-1 | Deity has highest margin, Bharatanatyam has highest volume |
| Trust signal | 0-2 | Reduces sizing, authenticity, shipping, or selection confusion |
| Regional vocabulary gap | 0-2 | Strong regional search terms lack coverage |
| EEAT and GEO citation value | 0-1 | Likely to become a concise source answer engines can cite |

Maximum score: 15.

## Default Tie-Breaker

When scores are close, use this priority order:

1. Collection support page
2. Buying guide
3. Size and fit guide
4. Seasonal guide or checklist
5. Comparison
6. Use case guide
7. Diaspora buying guide
8. Glossary page
9. History and explainer

Use judgment when seasonality is urgent. During May-August, Varalakshmi seasonal guides and checklists can outrank normally higher-priority formats.

## Angle Definitions

### Collection Support Page

Use when the topic should help buyers choose products inside an active collection.

Best for:

- high commercial intent
- strong internal linking
- collection SEO improvements
- collection snippets and FAQs

### Buying Guide

Use when buyers need selection help before purchase.

Best for:

- real kemp
- Bharatanatyam sets
- deity jewellery categories
- diaspora shoppers

### Size And Fit Guide

Use when fit is a blocker.

Best for:

- deity crown, mukut, kireedam
- idol jewellery by height
- kids Bharatanatyam jewellery
- waist belts, vaddanam, oddiyanam

### Seasonal Guide Or Checklist

Use when timing drives demand.

Best for:

- Varalakshmi
- arangetram preparation
- festival alankaram
- wedding styling

### Comparison

Use when buyers are confused between options.

Best for:

- real kemp vs regular Bharatanatyam jewellery
- kemp vs imitation kemp
- mukut vs kireedam
- oddiyanam vs vaddanam

### Use Case Guide

Use when a specific occasion drives the buying decision.

Best for:

- arangetram
- stage performance
- dance school bulk orders
- temple festival alankaram

### Diaspora Buying Guide

Use when the trust problem is online buying from outside India.

Best for:

- US, UK, Australia, Malaysia, Singapore, Europe customers
- international shipping
- authenticity and range explanation
- sizing and WhatsApp support

### Glossary Page

Use when regional vocabulary needs a durable internal-linking hub.

Best for:

- nethi chutti, mattal, oddiyanam, vanki, rakodi
- deity ornament names
- temple jewellery vocabulary

### History And Explainer

Use when the topic builds cultural context and answer-engine authority.

Best for:

- temple jewellery meaning
- kemp jewellery origins
- Bharatanatyam set components
