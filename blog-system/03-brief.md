# 03 Brief

Use this step after angle selection.

The goal is to create a clear content brief that a writer or AI assistant can draft from without becoming generic.

## Required Inputs

- topic intake
- research output
- angle selection output
- relevant wiki files
- `knowledge-snapshot.md`

## Internal Link Rules

Suggest links to:

- relevant collection pages
- relevant product pages
- glossary pages
- related buying guides
- size or fit guides

Use descriptive anchor text. Avoid vague anchors like `click here`.

## FAQ Rules

FAQs must answer real buyer decisions. Avoid generic questions.

Good:

- `How do I choose a real kemp set for arangetram?`
- `Is real kemp different from regular Bharatanatyam jewellery?`
- `Can Golden Collections ship Bharatanatyam jewellery to the USA?`

Weak:

- `What is jewellery?`
- `Why choose us?`

## Metadata Rules

The brief must include:

- SEO title
- meta description
- URL handle
- excerpt
- tags
- image alt text guidance

Keep metadata honest and aligned with the draft.
