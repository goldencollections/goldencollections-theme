# 05 EEAT Review

Use this checklist before creating a Shopify-ready package.

## Source And Claim Review

- [ ] Every factual claim has a wiki source, owner-confirmed source, Shopify source, live source, or cited research source.
- [ ] Uncertain facts are marked `Needs confirmation`.
- [ ] No certificate claims are present.
- [ ] No unsupported plating thickness claims are present for regular Bharatanatyam or Kuchipudi jewellery.
- [ ] Real kemp wording uses `Kemp stones` or `Kempu stones`.
- [ ] Material claims match the relevant product range.
- [ ] Shipping and delivery claims match owner-confirmed facts or live policy pages.

## Cultural Accuracy Review

- [ ] Ornament names are used correctly.
- [ ] Deity associations are correct.
- [ ] Ritual or festival context is accurate.
- [ ] Deity compatibility is not overstated.
- [ ] General/Common deity accessories are not described as universal if symbols, posture, deity, or size matter.
- [ ] Regional vocabulary is natural and relevant.

## Buyer Usefulness Review

- [ ] The introduction answers a real buyer situation.
- [ ] The piece helps a buyer choose, compare, size, care, or plan.
- [ ] There is at least one useful internal link.
- [ ] The FAQ block contains 4-6 specific buyer questions.
- [ ] The conclusion has a clear next action.
- [ ] The piece does not duplicate an existing published page.

## SEO And GEO Review

- [ ] H1 matches the content promise.
- [ ] SEO title is specific and honest.
- [ ] Meta description is complete.
- [ ] URL handle is clean.
- [ ] Excerpt is written.
- [ ] Tags are included.
- [ ] Image alt text guidance is included.
- [ ] Schema recommendation matches visible page content.
- [ ] FAQ schema is recommended only if the FAQ is visible on the page.
- [ ] Internal links use descriptive anchors.

## Owner Review Flags

Flag for owner review if:

- a material claim is not in the wiki
- a product range claim depends on current stock or live Shopify data
- a festival or ritual claim could be culturally sensitive
- a size rule is inferred rather than measured or owner-confirmed
- the content mentions returns, replacement, customs, or delivery in a way that should match live policies
