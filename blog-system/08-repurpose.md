# 08 Repurpose

Use this step after a Shopify draft exists or after the owner approves the final content direction.

The goal is to turn one strong content piece into platform-specific assets without making them feel copied.

## Output Location

Save repurpose files in:

`blog-system/outputs/repurpose/`

Use the same handle as the source content.

## Required Outputs

Create:

- Google Business Profile post
- Pinterest pin description
- Instagram caption
- YouTube Shorts or Reels script outline
- Email snippet
- WhatsApp broadcast message
- Quora or Reddit-style answer
- Collection page snippet

## Platform Rules

### Google Business Profile

- 150-300 words.
- Local trust angle is allowed: Golden Collections, Secunderabad/Hyderabad, specialist store.
- Include link to the content or collection.
- Keep it practical, not stuffed with keywords.

### Pinterest

- Keyword-rich, visual, and specific.
- Include regional terms naturally.
- Mention the product family and use case.

### Instagram

- Cultural and visual hook.
- Shorter paragraphs.
- Include regional terms naturally.
- Use `link in bio` if direct linking is not available.

### YouTube Shorts Or Reels

- 30-60 seconds.
- Show-not-tell format.
- Include shot list, voiceover, and on-screen text.
- Focus on one buyer question.

### Email

- Subject line.
- 3-4 sentence teaser.
- Clear CTA.

### WhatsApp Broadcast

- Short, direct, and useful.
- Include link.
- Avoid sounding like mass spam.

### Quora Or Reddit-Style Answer

- Informational framing first.
- Mention Golden Collections naturally only when relevant.
- Do not make it read like an advertisement.
- Answer the question genuinely.

### Collection Page Snippet

- 2-3 sentences.
- Useful for collection description or intro.
- Include regional terms only if natural.
