# Golden Collections SEO/GEO Knowledge Base

This folder is the working knowledge base for Golden Collections SEO, GEO, and answer-engine visibility.

## Workflow

1. Put raw source notes in `raw/`.
2. Compile stable business knowledge into `wiki/`.
3. Put audits, roadmaps, and generated reports in `outputs/`.
4. Use `ops/` for current operating memory: goals, decisions, open loops, source map, and owner briefs.
5. When new facts are learned, update the smallest relevant wiki page and backlink related pages.
