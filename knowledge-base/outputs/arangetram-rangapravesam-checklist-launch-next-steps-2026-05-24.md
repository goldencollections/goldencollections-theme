## Why Education First, Then Partner

The institutes already know the dance side. The first message should respect that by offering a useful parent checklist that supports the teacher's own list.

Partnership should come after permission is earned. If a teacher replies positively, then move to:

- teacher version of the checklist,
- component-list review for families,
- bulk/institute support,
- program sponsorship,
- partner code or teacher referral flow.

This avoids sounding like Golden Collections is asking for commercial access to students before proving value.
